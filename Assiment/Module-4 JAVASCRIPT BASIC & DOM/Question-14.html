Question:-14 What are all the types of Pop up boxes available in JavaScript?

ANS:-

  • There are three types of pop up boxes available in JavaScript.

    • 1.alert box:-

  • the alert box is used to display a simple message on display.
  • user can read and just give ok response.

    • 2. Confirm Box:-

  • the confirm is used to get two types response "OK" or "Cancel."

    • 3. Prompt Box:-

  • The prompt is used to get input from the user.
