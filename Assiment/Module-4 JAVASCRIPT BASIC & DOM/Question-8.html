Question:-8 What is === operator?

ANS:-

  • it is sign of strict equality.
  • it is relaational operator.
  • this operator compare value and data type.if both are equal then condition true otherwise false.
  • for example:-
  • document.write(bhakti === "bhakti")
    • false
  • console.log(bhakti === bhakti)
    • true
  • console.log("bhakti" === "bhakti")
    • true
