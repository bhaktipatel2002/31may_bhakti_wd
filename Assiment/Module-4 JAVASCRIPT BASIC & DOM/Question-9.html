ANS:-

By javascript , 2 method we can change class of element ,hear you cam see example of both method.

1. toggle method

  • Clicking the "toggle" button will toggle the red class on and off using classList.toggle('blue').

    • 2. replace method

  • Clicking the "Replace" button will replace the yellow class with the coral class using classList.replace('yellow', 'coral').
