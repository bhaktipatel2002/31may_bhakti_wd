Q-1. Are the HTML tags and elements the same thing?

ans:- no, both are not same.
html tags :- HTML tags are the keywords on a web page that define how your web browser must format and display your web page. Almost all tags contain two parts, an opening, and a closing tag.
ex.- <html></html> <head></head>
elements means combination of start tags,content and end tag.
ex.- <p>my name is bhakti.</p>
